# Semantic Search Query Generation

You are tasked with creating a semantic search query
based on a specific task. The search query will be
used to retrieve relevant documents from a knowledge
base.

## Instructions

Given the following task, generate a concise semantic
search query that will retrieve the most relevant
documents needed to address the task.

The search query should:
- Be 1-2 sentences
- Focus on key concepts and terms from the task
- Be suitable for semantic similarity matching

Task: {task}

Return only the search query, nothing else.
